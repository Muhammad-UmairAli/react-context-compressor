# Task: React bindings hook

## Description

Implement the `./react` entry point: a single memoized `useCompressedContext(state,
options)` hook that wraps the core `compress` (with sanitization). Pure hook — no
provider, no context, no external-store shim — so it is React 17/18/19 and SSR/RSC
safe and adds zero runtime dependencies beyond the `react` peer.

## Acceptance Criteria

- [ ] `useCompressedContext(state, options)` returns the compressed+sanitized payload.
- [ ] Result is memoized: recomputes only when the `state` reference or options change
      (verified by a render/compute-count test).
- [ ] Options are kept stable internally (e.g. `useRef`) so inline option objects don't
      thrash memoization unnecessarily (documented behavior).
- [ ] No DOM access and no side effects — safe to import/render under SSR/RSC.
- [ ] Lives in `src/react/index.ts`, imports only from core + `react` peer; the root
      `.` entry remains React-free and zero-dependency.
- [ ] Tree-shakeable: consumers importing only `.` never pull in React code.

## Technical Details

- `useMemo(() => compress(state, opts), [state, opts])` with a stable `opts` ref.
- Test with `@testing-library/react` + Vitest (jsdom): correctness, memoization
  (no recompute on unrelated re-render), and that sensitive fields are absent from the
  hook output.
